# Examples

这个目录用于放仓库展示层面的推理与调用样例。

推荐放入的内容包括：

1. 一到三条代表性题目
2. 模型输出的 reasoning
3. `\boxed{}` 最终答案
4. 与标准答案的对比
5. 哪一类题表现更好，哪一类题仍然容易错

## 当前文件

- [`inference_example.md`](inference_example.md): 推理展示模板

## 推荐补充方式

如果你后面跑通了 adapter 推理，可以直接把典型案例贴到这里，最好覆盖：

- Roman numeral conversion
- gravity conversion
- binary operation conversion
- character equation conversion
